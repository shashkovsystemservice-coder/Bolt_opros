# Bolt_opros

**Bolt_opros** - это современная платформа для создания, проведения и анализа опросов. Она разработана как гибкий инструмент, который позволяет быстро создавать как простые анкеты, так и сложные методологические опросники для профессиональных исследований.

### Ключевые возможности

- **Множество способов создания:**
    - **Ручной конструктор:** Интуитивно понятный интерфейс для создания опросов с нуля, добавления секций и различных типов вопросов.
    - **AI-генерация:** Быстрое создание проекта опроса на любую тему с помощью интеграции с Gemini AI.
    - **Каталог готовых методик:** Используйте проверенные стандарты и опросники для самодиагностики из встроенной библиотеки.
    - **Импорт из Excel:** Загружайте структуру опроса прямо из вашего файла.
- **Удобство для респондентов:**
    - **Понятный интерфейс:** Чистый и минималистичный дизайн, который помогает пользователям сфокусироваться на вопросах.
    - **Адаптивность:** Опросы отлично выглядят и легко проходятся на любых устройствах, от десктопов до смартфонов.
- **Сбор и экспорт данных:**
    - **Сбор информации:** Возможность собирать данные о респондентах перед началом опроса.
    - **Экспорт результатов:** Выгрузка индивидуальных результатов прохождения в форматах **PDF** и **Excel**.
- **Администрирование и безопасность:**
    - **Панель управления:** Удобный дашборд для управления шаблонами опросов, запусками и просмотра статистики.
    - **Безопасность:** Встроенная система аутентификации, защита роутов и управление доступом.

### Технологический стек

- **Фронтенд:** React, TypeScript, Vite, Tailwind CSS
- **Бэкенд и база данных:** Supabase
- **AI-интеграция:** Google Gemini
- **Деплой:** Firebase Hosting

### Быстрый старт

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/shashkovsystemservice-coder/Bolt_opros.git
    cd Bolt_opros
    ```

2.  **Установите зависимости:**
    ```bash
    npm install
    ```

3.  **Настройте переменные окружения:**
    Создайте файл `.env` в корневой директории проекта и добавьте в него ключи для доступа к Supabase и другим сервисам.
    ```
    VITE_SUPABASE_URL=https://your-project-ref.supabase.co
    VITE_SUPABASE_ANON_KEY=your-supabase-anon-key
    GEMINI_API_KEY=your-gemini-api-key
    ```

4.  **Запустите проект:**
    ```bash
    npm run dev
    ```

После этого проект будет доступен по адресу `http://localhost:5173`.
